# Habitat Builder UI Styles

This directory contains the styles for the Habitat Builder UI application. These styles follow the guidelines and standards defined in the centralized Style Guide.

## Important Note

The definitive source for styling guidelines is now located at:  
`/components/builder-web/STYLE_GUIDE.md`

Please refer to this centralized style guide for:

- Color palette definitions
- Typography standards
- Layout guidelines
- Component styling patterns
- Responsive design breakpoints
- Accessibility standards

## Directory Structure

- `abstracts/`: Mixins, variables, and utility functions
- `base/`: Base styles, typography, and resets
- `components/`: Styles for specific components
- `layout/`: Styles for major layout sections
- `themes/`: Theme definitions and variations

When adding new styles, please ensure they comply with the standards defined in the centralized Style Guide.
